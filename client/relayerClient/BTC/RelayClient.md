#跨链区搬运客户端-源链为比特币

##搬运客户端原理
搬运客户端通过向跨链区提交必要的源链数据，使得跨链区可以对源链交易进行SPV验证


##Python版本客户端(开发环境)

第三方库
```
web3.py >= 6
```
搬运客户端主要包含三个组件：源链接口，跨链区接口，链下模块

###源链接口
若搬运工可通过RPC访问比特币客户端，则在配置文件后，运行命令
```
python Relayer.py True
```

否则运行命令
```
python Relayer.py False
```

###跨链区接口
在跨链区1.0版本，跨链区的目标链是支持EVM的本地测试链，RPC地址为127.0.0.1，端口为8545。搬运客户端通过foundry提供的forge create和cast send命令与测试链进行交互。
