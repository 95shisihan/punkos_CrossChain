# 磐古系统概述

磐古的设计旨在使用模块化的区块链架构来构建一个去信任化的新型分布式操作系统。它将充当Web2和Web3之间的桥梁，使任何人都可以轻松地参与到互信计算和事务交互中来，为构建更加透明、安全、可靠、高效的数字世界奠定基础。

磐古系统的设计愿景总结如下。
- 兼容性：系统将兼容现有的主流操作系统、Web2生态和目前区块链主流生态，便于各类应用和服务进行迁移。
- 无服务器：系统基于区块链技术，不依赖任何中心化服务器，确保系统的自主性和可靠性，同时使用户能够更轻松自由地加入到系统生态中。
- 可验证：系统中所有计算与事务交互过程可验证，并配合系统激励建立透明公平的交互机制，避免欺诈和不当行为并确保参与者的利益。
- 可演化：系统从功能上进行解耦，通过模块化架构为系统演化、系统升级提供可能。系统将通过良好的社区生态构建，以社区治理的方式实现技术、功能和生态不断升级更新。
- 可持续：系统参考现有区块链中的激励机制，提供与现今环境更符合的激励机制以鼓励系统用户参与系统生态建设和系统治理决策，激发参与者的积极性和创造力。
